Documentation page

This is a server-side component of a “movies” web application which is similar to Netflix. This web application will provide users with access to information about different movies as requested.

HTTP Method Request URL Request body data format Response body data format
GET Get all movie lists '/movies' None Get data array of all movie lists in a JSON format
GET Get data about a single movie title '/movies/:Title' None Get an array of a single movie data in JSON format, based on title entered in the URL parameter. Example: { Title:"Inglorious Basterds", Description:"A few Jewish soldiers are on an undercover mission to bring down the Nazi government and put an end to the war. Meanwhile, a woman wants to avenge the death of her family from a German officer.", Genre: { Name:"Action", Description:"Action film is a film genre in which the protagonist is thrust into a series of events that typically involve violence and physical feats.", }, Director: { Name:"Quentin Tarantino", Bio:"Quentin Jerome Tarantino is an American filmmaker, film director, screenwriter, producer, film critic, and actor. His films are characterized by nonlinear storylines, dark humor, stylized violence, foot fetishism, extended dialogue, ensemble casts, references to popular culture, alternate history, and neo-noir.", Birth:"1963-03-27" }, Actors: ['Brad Pitt', 'Eli Roth', 'Diane Kruger'], ImagePath:"https://www.imdb.com/title/tt0361748/mediaviewer/rm3163035648/?ref_=tt_ov_i", Featured: false, yearOfRelease: 2009 }
GET Get movie by genre '/genres/:genreName' None Get an array of movie by genres plus the genre description in a JSON format
GET Get data about a movie director 'directors/:directorName' None Get an array of movie director/directors data in a JSON format, based on the director entered in the URL parameter: Example { Director: { Name:"Tatsuo Sato", Bio:"Tatsuo Satō is a Japanese anime director most famous for Martian Successor Nadesico.", Birth:"1964-05-07" Death: if applicable }, }
GET Allow for the list of users '/users' user information is returned from the server in a form style A text message of user information, included with a unique id and an extra success message. Example { "_id" : ObjectId("61fda65a986730c619143ec3"), "Username" : "dono321", "Password" : "donnate", "Email" : "dona@gmail.com", "Birthday" : "1992-01-01", "Favouritemovies" : [ ] }
POST Allow clients to create account '/users user information is added to the server in a form style A text message of the client's added information, included with a unique id and an extra success message. Example { "_id" : ObjectId("61fda65a986730c619143ec3"), "Username" : "newUser", "Password" : "Password", "Email" : "newUser@gmail.com", "Birthday" : "1998-01-01", "Favouritemovies" : [ ] }
PUT Allow clients to update their client info '/users/:Username' None A text is returned to confirm the updated information in this case the username
POST Allow clients to add their favourite movies to their library '/users/:Username/movies/:MovieID' An array of JSON data containing the added movie. Example: { title: 'Jungle Book', director: ['Jon Favreau'], genre: ['Family'], releaseYear: 2016 } An array of the added movie data in JSON format, with a text confirming added data and an added ID. Example: { id: 79445, title: 'Jungle Book', director: ['Jon Favreau'], genre: ['Family'], releaseYear: 2016 }
DELETE Allow clients to remove a movie from their list of favorites '/users/:Username/movies/:MovieID' None Returns a text confirming which movie that was actually deleted
DELETE Allow clients to terminate account '/users/:Username' None Returns a text confirming user deregistration
